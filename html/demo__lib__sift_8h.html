Apply initial smoothing to input image to raise its smoothing to par.InitSigma. We assume image from camera has smoothing of sigma = 0.5, which becomes sigma = 1.0 if image has been doubled. increase = sqrt(Init^2 - Current^2)

Convolve by par.InitSigma at each step inside OctaveKeypoints by steps of Subsample of factor 2 while reasonable image size
