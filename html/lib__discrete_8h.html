Detailed Description

simple image transformations

This is a front-end to libpng, with routines to:

  • Separable discrete convolutions
  • Gaussian blur via discrete convolution with truncated kernel
  • 2D Gradient
  • Subsampling by integer factor
  • bilinear interpolation
