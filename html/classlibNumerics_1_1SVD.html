Solve the linear system Ax = 0 with ||x|| = 1.0 via SVD.

Check if there is a single singular value such that SV/maxSV<ratioExtremes. However if the two lowest SV satisfy this, accept if minSV/minSV2<ratio2Min. This indicates whether the solution is unique.
