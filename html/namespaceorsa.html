Estimate the fundamental matrix using ORSA method and refinement.

ORSA fundamental matrix estimation.

If the mean error after refinement exceeds the max error of ORSA result, the refinement is not applied.

Parameters
[in]vec_matchingsList of correspondences.
[in]w1,h1Dimensions of left image.
[in]w2,h2Dimensions of right image.
[in]precisionMaximum inlier/outlier threshold (in pixels).
[in]nbIterMaximal number of iterations for RANSAC algorithm.
[out]FFundamental matrix between left and right image.
[out]vec_inliersIndex of inliers in vec_matchings.
Returns
The status of the estimation. If no meaningful (NFA<1) model is found, the F field is not filled and vec_inliers is empty.
